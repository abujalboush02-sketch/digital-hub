Workflow API Tester

Tester Checklist & Instructions

  1. Get Admin Access: Ask the developer for the raw (unhashed) Admin API Key (It must be manually inserted into the PostgreSQL ApiKeys table).
  2. Set Global Key: Paste the Admin Key into the Global X-Api-Key field in the top navigation bar.
  3. Create User Key: Go to the API Keys Management tab and create a new key with the role User. Note the new key generated.
  4. Test Webhook (No Auth): Go to the WhatsApp Simulator tab. Type a message (e.g., "I need 2 pizzas") and send it. Note the resulting WhatsAppMessage ID.
  5. Retrieve Results: Go to the Parsed Results tab. Use either your User Key or Admin Key to fetch the result using the ID from the previous step.
  6. Test Permissions: Swap your Global Key to the User key and try to delete/create an API key. It should return a 403 Forbidden.
